## What is set()?
``` python
# Create an empty set
empty_set = set()

# Create a set from a list
numbers = set([1, 2, 3, 2, 1])
print(numbers)  # Output: {1, 2, 3}
```
Duplicate values are automatically removed!   

✅ No duplicates

✅ Unordered (elements have no index)

✅ Mutable (you can add/remove items)
