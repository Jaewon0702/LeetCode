### What is this code?
```python
dummy = ListNode(0,head)
```
- 0 is the value of the dummy node (doesn’t really matter — we ignore it).  

- head is the actual input linked list.  

- So the dummy node points to the head.
```scss
dummy(0) → 1 → 2 → 3

```
